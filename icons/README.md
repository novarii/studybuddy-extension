# Icons Directory

This directory should contain the following icon files:

- `icon16.png` - 16x16 pixel icon
- `icon48.png` - 48x48 pixel icon  
- `icon128.png` - 128x128 pixel icon
- `icon.svg` - Vector icon (optional)

You can create these icons using any image editor or icon generator. The icons should represent the Panopto Video Downloader extension.
